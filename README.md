# bpsproject
스마트 인재 개발원 - 인공지능 융합 서비스 개발자 과정 <br>
- 핵심 프로젝트 <br>
- 리버스 엔지니어링 : BPS <br>
- 주제 : 사용자 맞춤형 업무관리 시스템 개발 <br>

## 프로젝트기간 <br>
- 2023.09.05 ~ 2023.09.21

## 주요기능 <br>
* api 로그인을 활용한 회원가입 및 로그인 <br>
* 출퇴근 시간 체크 <br>
* 실시간 채팅 기능 <br>
* 캘린더 일정 등록 <br>
* 차량 정보 등록 및 조회 <br>
* 차량 사용 신청 및 반납 기능 <br>
* 드라이브 파일 등록 및 삭제 <br>
* 게시판 글 작성
